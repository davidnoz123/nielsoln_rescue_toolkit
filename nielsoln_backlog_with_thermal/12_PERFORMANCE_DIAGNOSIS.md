# PERFORMANCE_DIAGNOSIS

Implement cross-module performance diagnosis.

Correlate:
- disk type and health
- storage fullness
- RAM amount and pressure
- CPU generation / capability
- thermal throttling
- startup entries
- bloatware/software inventory
- services load

Answer:
- why is the machine slow?
- what gives the best improvement?
- what is not worth upgrading?

Output:
- `logs/performance_diagnosis_<timestamp>.json`
