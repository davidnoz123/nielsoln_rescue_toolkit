# System Health Prompt Library

This ZIP contains Claude-ready prompts for extending the RescueZilla/Linux triage tool into a hardware health, software cleanup, upgrade-advisory, and client-reporting system.

Each module has its own Markdown file. Copy the fenced prompt text from a file into Claude when you want that module implemented.

Recommended starting sequence:

1. DISK_HEALTH
2. HARDWARE_PROFILE
3. UPGRADE_ADVISOR
4. SOFTWARE_INVENTORY
5. SYSTEM_SUMMARY

The prompts are intentionally high-level. They describe behaviour, safety, scope, and deliverables without forcing low-level implementation details.
